# Security Policy

## Supported Versions

MC-ELMA Rope `0.4.0-beta.2` is the current supported beta line.

## Reporting a Vulnerability

Report suspected security issues privately to the project owner before opening a
public issue when the report includes exploit details, server compromise risk,
or private server information.

Public GitHub issues are appropriate for ordinary gameplay bugs that do not
expose credentials, private infrastructure, or player personal data.

Do not include server IPs, tokens, private logs, whitelist files, ops files, or
RCON credentials in public reports.
